# Smart Facelift Platform
An AI-powered marketplace for aesthetic and tech home upgrades.

## Tech Stack
- **Frontend:** Next.js (Deployed to Azure Static Web Apps)
- **Backend:** Node.js / Azure Functions (Serverless Bidding Logic)
- **Database:** PostgreSQL on Azure (Prisma ORM)
- **Storage:** Azure Blob Storage (High-speed Gallery)

## Getting Started
1. `npm install` in the root.
2. `npx prisma generate` in `packages/database`.
3. `npm run dev` in `apps/web`.
